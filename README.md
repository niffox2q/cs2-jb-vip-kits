### Модуль VIP Kits для Jailbreak Core  

Добавляет систему Наборов для VIP игроков.

В конфиге можно настроить получаемое оружие, гранаты.
Есть возможность выключить / включить наборы во время Выходного и Игрового дня.
Предусмотрена систему кулдаунов между наборами, например раз в 5 раундов доступно взять набор, для каждого набора есть возможность выставить кулдаун. Он распространяется на все наборы.

Для каждого оружия есть возможность выставить кастомные параметры (для оружия - настроить обоймы, для гранаты - наносимый урон).
Более подробно описано в конфиге.

### Установка:
Распаковать в `game/csgo/addons/`
